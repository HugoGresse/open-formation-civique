**Dernière mise à jour : février 2026**

Le site **open-formation-civique.fr** est conçu dans le respect de votre vie privée. Cette page détaille les données traitées par le site.

## Responsable du traitement

**Hugo Gresse**
Contact : hugo.gresse [at] gmail.com

## Cookies

**Ce site n'utilise aucun cookie.**

Aucun cookie de suivi, publicitaire, ou de fonctionnement n'est déposé sur votre navigateur.

## Mesure d'audience

Ce site utilise [Plausible Analytics](https://plausible.io/), un outil de mesure d'audience auto-hébergé, respectueux de la vie privée.

Plausible :
- **Ne dépose aucun cookie**
- **Ne collecte aucune donnée personnelle**
- Ne permet pas le suivi entre différents sites
- Ne crée pas de profil utilisateur
- Est conforme au RGPD sans nécessiter de consentement ([exempté par la CNIL](https://www.cnil.fr/fr/cookies-et-autres-traceurs/regles/cookies-solutions-pour-les-outils-de-mesure-daudience))

Les seules données collectées de manière anonyme sont : la page visitée, le référent (site d'origine), le type d'appareil et le pays. Ces données sont agrégées et ne permettent pas d'identifier un visiteur.

## Stockage local (localStorage)

Lorsque vous utilisez les quiz, votre progression et vos résultats sont enregistrés **localement dans votre navigateur** (localStorage) afin de vous permettre de reprendre un quiz en cours ou de consulter vos scores.
Pas de login ni de compte utilisateur = pas de données personnelles. Il y a des désavantages, mais au moins tout le monde est protégé !

Ces données :
- **Restent uniquement sur votre appareil**
- Ne sont jamais transmises à un serveur
- Ne contiennent aucune information personnelle identifiante
- Peuvent être supprimées à tout moment via le bouton de réinitialisation des quiz ou en effaçant les données de votre navigateur

## Données personnelles collectées

**Aucune donnée personnelle n'est collectée par ce site.**

Il n'y a pas de formulaire de contact, de création de compte, de newsletter, ni de système d'authentification.

## Services tiers

Le seul service externe chargé par ce site est le script Plausible Analytics (auto-hébergé sur plausible.gresse.io). Aucun autre service tiers (Google, Facebook, réseau publicitaire, etc.) n'est utilisé.

## Vos droits

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :
- **Droit d'accès** : connaître les données vous concernant
- **Droit de rectification** : corriger des données inexactes
- **Droit à l'effacement** : demander la suppression de vos données
- **Droit à la portabilité** : recevoir vos données dans un format structuré

Étant donné qu'aucune donnée personnelle n'est collectée côté serveur, ces droits s'appliquent uniquement aux données stockées localement dans votre navigateur, que vous pouvez consulter et supprimer librement.

Pour toute question, contactez : hugo.gresse [at] gmail.com
